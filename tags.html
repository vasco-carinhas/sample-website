Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header space creates a header space
div division space creates a division space
ol ordered list creates a numbered lis
ul unordered list creates a bulleted list
li list item creates an item within a list
blockquote block quotation show formatted quotation
!- comments tag hides text for commenting

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
i italic makes text italic <i>technical sophistication</i> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
b bold makes text bold <b>at least a billion people</b> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code style as computer code <code>This reads like code.</code> This reads like code.
span span mark part of text <span>This text is marked.</span> This text is marked.
br break enter line break These two lines <br> have a break tag between them. These two lines
have a break tag between them.
a anchor interconnect two pages Link to: <a href="moby_dick.html"> Moby Dick Book Report </a> Link to: Moby Dick Book Report

Document Tags

Tag Name Purpose
html html root of an HTML document and the container of most of its elements
head head container for metadata
body body defines the document's body
title title defines the document's title
meta meta defines document's metadata
style style defines document's styling

Not Really a Tag

Tag Name Purpose
!DOCTYPE document type not a tag, but a declaration of the version of HTML of the document
